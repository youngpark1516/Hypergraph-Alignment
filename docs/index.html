Wasserstein Hypergraph Learning for General 3D Point Cloud Alignment

Chanyoung Park (chp026@ucsd.edu) · Minghan Wu (miw039@ucsd.edu) · Gal Mishne (gmishne@ucsd.edu) · Yusu Wang (yuw122@ucsd.edu)

We improve hypergraph-based 3D alignment by replacing mean pooling with a Wasserstein-based aggregation that better preserves local feature structure.

Code Report (PDF)

Overview

Accurate 3D point cloud alignment is used for shape correspondence, reconstruction, and tracking. In practice, alignment is hard because of non-rigid deformation, partial observations, and noise.

Our project focuses on a specific part of hypergraph learning pipelines: how we summarize neighborhood (hyperedge) information. Instead of averaging (mean pooling), we use a distribution-aware aggregation based on optimal transport ideas.

Why this matters

Many alignment systems depend on local neighborhoods being represented well enough to disambiguate correspondences. When neighborhoods are multi-modal or sampling density changes, simple averaging can blur important structure and reduce robustness.

Optional: one-paragraph technical context

Hypergraph models represent higher-order relationships (groups of points), which can better capture local geometric consistency than pairwise edges alone. The choice of aggregation inside these hypergraph layers can influence both accuracy and robustness.

Approach (high-level)

Start from a hypergraph alignment baseline

We build on a dynamic hypergraph pipeline for point cloud alignment that groups candidate correspondences into hyperedges and refines them through message passing.

Replace mean pooling with Wasserstein aggregation

Instead of summarizing a neighborhood by a single average vector, we treat the set of neighborhood features as a distribution and aggregate in a way that is sensitive to how that “mass” is arranged in feature space.

Provide a matching-based hypothesis generation step

We also implement a matching-based hypothesis generation procedure as an alternative to Iterative Closest Point (ICP) for piecewise-rigid settings.

Evaluation (high-level)

We evaluate on standard 3D datasets under multiple perturbation settings. We report correspondence quality using standard classification metrics, and we also use structural diagnostics to understand hypergraph quality and robustness behavior.

What we measure (placeholder)
  • Task performance: precision / recall / F1 (plus losses)
  • Hypergraph structure: diagnostics that reflect grouping and separation quality
  • Robustness: how results change across perturbation regimes

Results & takeaways

Overall, our results indicate that Wasserstein-based aggregation can improve performance over mean pooling and can change robustness behavior across perturbation regimes.

  • Takeaway 1: Distribution-aware aggregation can preserve local structure that averaging loses.
  • Takeaway 2: Where you insert Wasserstein aggregation (early vs. late) can matter.
  • Takeaway 3: Robustness behavior can shift across perturbation settings.
